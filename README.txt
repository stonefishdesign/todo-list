To-do List - Created by Megan Li (http://stonefishworks.com)

This is a very simple responsive web application that allows user to create, edit, save and delete short textual items. 

To add note, simply type in the input field and hit Enter or click on the Save button. Blank item will not be saved.

To edit note, hover over or click (mobile browsers) on the item and click the Edit button.

To delete note, hover over or click (mobile browsers) on the item and click the Delete button. All notes can be purged at once using the Delete All button on the top left.

Information is save in browser local storage.
